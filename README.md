# WEB SCRAPING USING SCRAPY

Scraped jumia `https://www.jumia.com.gh/` products and seller<br>
information and store in sqlite database.

### ITEMS SCRAPED

- **Seller Details**
    - Name
    - Country of Origin
    - Order Fulfillment Rate
    - Quality Score
    - Seller Score
    - Followers
- **Product URL**
- **Product Name**
- **Brand**
- **Product description**
- **Price**
- **Discounted Price**
- **Discount**
- **Image URL**
- **Customer Reviews**
- **Ratings**
- **Category**
